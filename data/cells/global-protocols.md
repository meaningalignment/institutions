# ISO standards & internet protocols (TCP/IP, HTTP)

## At a glance

### Coordination challenge

How the world settles on shared technical standards no government or firm controls.

### Examples

- TCP/IP
- HTTP
- ISO standards
- Unicode
- GSM cellular standards

### How AGI breaks them

- De facto standards are set by the largest model deployments.
- Global consensus bodies move years; agents iterate weekly.
- Spec compliance is unverifiable across closed model stacks.
- No neutral arbiter for cross-vendor agent interop.
