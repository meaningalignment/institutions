# Deliberative value elicitation for group decisions

When a group decides — on a policy, a direction, a contested call — the standard mechanism is vote or consensus, both of which compress rich deliberation into a thin summary. A fidelity-oriented process instead surfaces *what participants value*, with the context and tradeoffs behind each position, and distinguishes articulations that reflect deeper engagement with the domain from those that reflect idiosyncratic preference.

Mechanisms in this family:

- Deliberative polling (Fishkin)
- Citizens' assemblies and policy juries
- Pol.is–style consensus discovery
- Moral Graph Elicitation, which uses structured interviews and peer-rated depth judgments
- Quadratic voting with justification

Design questions:
- How do you resist both populist capture and expert capture?
- How should elicitation interface with the decision itself — binding, advisory, presumptive?
- How does the process handle values that *change* through it, which is often the mark of good deliberation working?
- What's the right participant-count range for a given group's decision? Too few and the result isn't representative; too many and deliberation thins.
